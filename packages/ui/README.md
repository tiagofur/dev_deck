# @devdeck/ui

The official **DevDeck Design System**. A collection of reusable, neo-brutalist inspired UI components built with React, Tailwind CSS, and Framer Motion.

## Aesthetic

DevDeck uses a **Neo-brutalist / High-contrast Dark Mode** aesthetic:
- Thick black borders (`border-2 border-black`)
- Vibrant accents (Vivid Blue, Cyber Lime)
- Sharp shadows (`shadow-[4px_4px_0px_0px_rgba(0,0,0,1)]`)
- Heavy typography (Inter/Outfit)

## Components

- **Buttons:** Various styles (Primary, Ghost, Outline).
- **Inputs:** Themed text inputs and textareas.
- **Feedback:** Toasters, Modals, and Status Badges.
- **Cards:** The base container for almost everything in DevDeck.

## Setup

This package exports a **Tailwind Preset** that must be included in your `tailwind.config.js` to get the correct colors and spacing tokens.
