# @devdeck/api-client

The official SDK for interacting with the DevDeck Go API.

## Features

- **Typed Clients:** Auto-generated-like types for all API responses.
- **TanStack Query Hooks:** Pre-configured hooks for data fetching (`useRepos`, `useCheatsheets`, etc.).
- **Auth Interceptors:** Automatically handles JWT inclusion and Refresh Token rotation.
- **Multimodal Adapters:** Different configurations for Web (Cookies) vs Desktop (Secure Header).
